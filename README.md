React Testing App (migrated contact page)

This folder contains a Vite + React application that reproduces the UI found at htmlfiles/contact.html.

Getting started

1. Change into the project root and install dependencies:

   npm install

2. Start the dev server:

   npm run dev

3. Build for production:

   npm run build

Notes

- The original HTML source-of-truth remains at htmlfiles/contact.html and should not be modified.
- The React app mounts the migrated contact page at the route: /contact
